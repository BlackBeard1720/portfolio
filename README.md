# Minh Static Portfolio

A lightweight portfolio rebuilt with plain HTML, CSS, and JavaScript.

## Structure

```text
minh-static-portfolio/
├── index.html
├── style.css
├── script.js
└── assets/
    └── cv.pdf
```

## How to use

1. Open `index.html` directly in your browser.
2. Replace `assets/cv.pdf` with your real CV file.
3. Update project descriptions or links if needed.
4. Push to GitHub Pages.

## GitHub Pages deploy

Because this is plain HTML/CSS/JS, you can deploy directly from the repository:

1. Put `index.html`, `style.css`, `script.js`, and `assets/` in the root of the repo.
2. Go to GitHub repository → Settings → Pages.
3. Source: `Deploy from a branch`.
4. Branch: `main`, folder: `/root`.
5. Save.
